## **4个空格**

---

在每行前面加上4个空格

```text
	text = "Hello, world!"
	print text
```

效果

	text = "Hello, world!"
	print text

## **3个 `**

---

	```text
	text = "Hello, world!"
	print text
	```

效果

```text
text = "Hello, world!"
print text
```

**注1: 支持代码高亮**，详见"[区块代码高亮](./highlight_code.md#_2)"

**注2: 如果代码区块上方是列表，如[无序列表](./list_noseq.md)或[有序列表](./list_seq.md)，那么4个空格不会被解析为代码区块，而是被解析为一个普通缩进**
